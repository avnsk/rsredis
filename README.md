### Redis Lite

A lightweight Redis server clone built from scratch in Rust.

### Current Progress

*   Configured a TCP server listening on port `7379`.

### How to Run

1.  Start the server:
    
    bash
    
        cargo run
        
    
    Use code with caution.
    
2.  Test the connection in a separate terminal:
    
    bash
    
        nc 127.0.0.1:7379
        
    
    Use code with caution.
